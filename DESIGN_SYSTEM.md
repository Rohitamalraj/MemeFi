# MemeFi Design System

## Colors
- **Background**: `#000000` (Black)
- **Foreground**: `#ffffff` (White) 
- **Primary**: `#FFC700` (Yellow)
- **Border**: `#424242` (Dark Gray)

## Typography
- **Headings**: Sentient font (`font-sentient`)
- **Body**: JetBrains Mono (`font-mono`)

## Component Patterns
- Black backgrounds
- Yellow primary accents
- Clean, minimal cards with dark borders  
- Consistent button styling using primary yellow
- White text with opacity variations

This design is now applied across all pages for consistency.
